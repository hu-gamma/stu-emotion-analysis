# 消融实验报告：Baseline vs DropoutEnhanced

**对比模型**: V6a (dropout=0.1) vs V6a-DropoutEnhanced (RoBERTa/LSTM/Classifier dropout=0.3)

---

## 1. 整体性能对比

| 指标 | Baseline (dropout=0.1) | DropoutEnhanced (dropout=0.3) | 变化 |
|------|------------------------|-------------------------------|------|
| **测试集 Accuracy** | 66.59% | 64.72% | -1.87% |
| **测试集 Macro F1** | **0.5703** | **0.5627** | -0.0076 |
| **测试集 Weighted F1** | 0.6636 | 0.6537 | -0.0099 |
| **训练集 Accuracy** | 84.50% | 80.02% | **-4.48%** |
| **训练集 Macro F1** | 0.8677 | 0.8281 | **-0.0396** |
| **Train-Test Accuracy Gap** | 17.91% | 15.30% | **-2.61%** |
| **Train-Test Macro F1 Gap** | 0.2974 | 0.2654 | **-0.0320** |
| **Silhouette Score** | 0.0427 | **0.0648** | **+0.0221** |

### 结论
- **DropoutEnhanced 测试性能略有下降**（Macro F1 从 0.5703 → 0.5627，-1.3%）
- **但过拟合程度显著减轻**：Train-Test F1 gap 从 0.297 → 0.265（改善 10.8%）
- **特征空间分离度提升**：Silhouette Score 从 0.043 → 0.065（+52%），说明各类别在特征空间中更分散

## 2. 各类别过拟合对比

| 类别 | Baseline F1 Gap | Dropout Gap | 改善 |
|------|-----------------|-------------|------|
| 悲伤 | +0.1273 | **+0.0909** | **-28.6%** |
| 快乐 | +0.0995 | **+0.0834** | **-16.2%** |
| 愤怒 | +0.3367 | **+0.2389** | **-29.0%** |
| 焦虑 | +0.2935 | **+0.2465** | **-16.0%** |
| 尴尬 | +0.4471 | +0.4432 | -0.9% |
| 厌恶 | +0.4170 | **+0.3813** | **-8.6%** |
| 惊讶 | +0.4094 | **+0.3913** | **-4.4%** |
| 好奇 | +0.2485 | **+0.2473** | **-0.5%** |

### 结论
- **悲伤、快乐、愤怒、焦虑**的过拟合差距显著缩小（改善 16-29%）
- **尴尬**几乎无改善（0.447→0.443），说明样本量太少（153条），仅靠 dropout 无法解决
- **惊讶**改善有限（0.409→0.391），同理样本量仅181条

## 3. 错分模式对比

| 排名 | Baseline 错分流 | 比例 | Dropout 错分流 | 比例 | 变化 |
|------|----------------|------|---------------|------|------|
| 1 | 愤怒 → 悲伤 | **31.1%** | 愤怒 → 悲伤 | **20.3%** | **-10.8%** |
| 2 | 尴尬 → 悲伤 | **26.8%** | 尴尬 → 悲伤 | **20.3%** | **-6.5%** |
| 3 | 焦虑 → 悲伤 | **20.9%** | 惊讶 → 快乐 | 19.9% | - |
| 4 | 惊讶 → 快乐 | 20.4% | 悲伤 → 愤怒 | **15.6%** | 新出现 |
| 5 | 厌恶 → 悲伤 | **19.9%** | 厌恶 → 愤怒 | **14.2%** | -5.7% |

### 关键变化
- **"悲伤吸星"效应大幅减轻**：愤怒→悲伤 从 31.1% 降到 20.3%，尴尬→悲伤 从 26.8% 降到 20.3%
- **悲伤→愤怒**成为新的错分模式（15.6%），说明模型对愤怒和悲伤的边界在调整
- **愤怒召回率提升**：从 41.5% → 52.8%（+11.3%），更多愤怒样本被正确识别

## 6. 综合评估

### DropoutEnhanced 的优点
1. **过拟合显著减轻**：Train-Test gap 缩小 ~11%
2. **特征空间分离度提升**：Silhouette +52%
3. **"悲伤吸星"效应减弱**：愤怒/尴尬/焦虑 错分为悲伤的比例明显下降
4. **愤怒召回率提升**：从 41.5% → 52.8%

### DropoutEnhanced 的缺点
1. **整体测试性能略降**：Macro F1 -0.0076（约 1.3%）
2. **小样本类别过拟合依然严重**：尴尬/惊讶 gap 仍 > 0.39
3. **Val loss 持续上升**：说明 dropout=0.3 仍不足以阻止记忆

### 关键洞察
> 增加 dropout 确实缓解了过拟合（gap 缩小）和类别混淆（悲伤吸星减弱），但代价是整体性能略微下降。这说明**模型欠拟合和过拟合并存**——小样本类别过拟合（尴尬/惊讶），而大样本类别（悲伤/快乐）可能开始欠拟合。

### 下一步建议
1. **不对称 dropout**：对 BiLSTM 和 Classifier 用高 dropout (0.3-0.5)，但对 RoBERTa 保持较低 (0.1)，避免破坏预训练特征
2. **小样本专用增强**：对尴尬/惊讶/厌恶做回译/同义词增强，而非仅靠 dropout
3. **分层正则化**：对大类别（悲伤/快乐）降低权重，对小类别提高权重，配合 Focal Loss gamma=3.0
4. **尝试 Label Smoothing**：eps=0.1，让模型对"悲伤"不那么自信
