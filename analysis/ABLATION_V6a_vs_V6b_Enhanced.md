# 消融实验报告：V6a (Baseline) vs V6b-Enhanced (GoEmotions数据增强)

**对比模型**:
- **V6a**: 原始 `split_train.csv` (41,310条) + Focal Loss + dropout=0.1
- **V6b-Enhanced**: `split_train_enhanced.csv` (43,358条) = 清洗后数据 + GoEmotions翻译补充(厌恶+1,500条, 惊讶+600条) + Focal Loss + dropout=0.1

**测试集**: `split_test.csv` (9,951条, 8类)

---

## 1. 整体性能对比

| 指标 | V6a (Baseline) | V6b-Enhanced | 变化 |
|------|----------------|--------------|------|
| **测试集 Accuracy** | 66.59% | 66.52% | -0.07% |
| **测试集 Macro F1** | 0.5703 | **0.6058** | **+0.0355** |
| **测试集 Weighted F1** | 0.6636 | **0.6711** | **+0.0075** |
| **验证集最佳 Macro F1** | 0.5779 | **0.6520** | **+0.0741** |
| 推理延迟 | 1.67 ms | 1.67 ms | 持平 |

### 关键发现

- **Macro F1 显著提升**：从 0.570 → 0.606 (+6.2%)，突破 0.60 门槛
- **验证集提升更大**：Val Macro F1 从 0.578 → 0.652 (+12.8%)，说明模型泛化能力增强
- **Accuracy 基本持平**：-0.07%，说明提升主要来自小类的识别改善
- **推理速度不变**：模型架构完全相同，速度无变化

## 3. 混淆矩阵对比

### 厌恶 (Class 5) 混淆分析

| 预测方向 | V6a 比例 | V6b 比例 | 变化 |
|---------|---------|---------|------|
| → 厌恶 (正确) | 45.6% | **78.8%** | **+33.2%** |
| → 悲伤 | 19.9% | **4.1%** | **-15.8%** |
| → 好奇 | 11.9% | **2.1%** | **-9.8%** |
| → 愤怒 | 9.6% | 10.4% | +0.8% |
| → 快乐 | 7.8% | **2.1%** | **-5.7%** |

### 惊讶 (Class 6) 混淆分析

| 预测方向 | V6a 比例 | V6b 比例 | 变化 |
|---------|---------|---------|------|
| → 惊讶 (正确) | 52.5% | **76.2%** | **+23.7%** |
| → 快乐 | 20.4% | **9.4%** | **-11.0%** |
| → 好奇 | 13.3% | **2.2%** | **-11.1%** |
| → 愤怒 | 5.5% | 6.1% | +0.6% |
| → 悲伤 | 6.1% | 5.0% | -1.1% |

### 关键发现

- **"悲伤吸星"效应在厌恶/惊讶上显著缓解**：
  - 厌恶→悲伤：19.9% → 4.1% (-15.8%)
  - 惊讶→快乐：20.4% → 9.4% (-11.0%)
- **模型学会了区分厌恶和好奇**：厌恶→好奇 11.9% → 2.1%
- **愤怒→悲伤仍然严重**：31.1% → 未显著改善（数据清洗的边际效益）

## 6. 与 Qwen2.5-3B 对比

| 指标 | RoBERTa-BiLSTM v6a | RoBERTa-BiLSTM v6b | Qwen2.5-3B (LoRA) |
|------|-------------------|-------------------|-------------------|
| Accuracy | 66.59% | 66.52% | **68.28%** |
| **Macro F1** | 0.5703 | **0.6058** | 0.5883 |

| 推理延迟 | 1.67 ms | 1.67 ms | 734 ms |
| 参数量 | 110M | 110M | 3B |

### 关键结论

- **v6b Macro F1 (0.606) 超过 Qwen (0.588)**：在轻量级模型上实现了比大模型更高的细粒度识别能力
- **v6b 速度优势 440 倍**：1.67ms vs 734ms，完全满足实时推理需求
- **Qwen 在 Accuracy 和 Weighted F1 上仍领先**：说明 Qwen 在大类（悲伤/快乐）上有优势，但小类识别不如 v6b

## 7. 综合评估

### V6b-Enhanced 的改进

1. **突破 0.60 Macro F1 门槛**：从 0.570 → 0.606，提升 6.2%
2. **厌恶识别翻倍**：F1 从 0.50 → 0.70，识别率从 46% → 79%
3. **惊讶识别大幅提升**：F1 从 0.49 → 0.58，识别率从 53% → 76%
4. **愤怒 Recall 提升**：0.415 → 0.566，标签清洗见效
5. **验证集泛化能力增强**：最佳 Val F1 0.652 vs 0.578
6. **超越 3B 大模型**：Macro F1 0.606 > Qwen 0.588

### V6b-Enhanced 的局限

1. **尴尬仍为短板**：F1 0.44，无增强数据，是新的瓶颈
2. **悲伤/快乐略有下降**：注意力被小类"分散"
3. **GoEmotions 翻译腔**：英文 Reddit 翻译文本与中文社交媒体风格存在差异

### 核心洞察

> **数据增强策略（P0厌恶 + P1惊讶）非常成功**：
> - 补充的小类数据直接解决了模型最困惑的混淆方向
> - 厌恶→悲伤错分从 20% 降至 4%，惊讶→快乐错分从 20% 降至 9%
> - 翻译数据的 domain shift 风险被控制在可接受范围内

---

## 8. 下一步建议

1. **尴尬数据增强（P2）**：从 GoEmotions 补充尴尬类翻译数据，或尝试 LLM 生成更高质量的尴尬样本
2. **继续愤怒标签清洗**：当前仅清洗了训练集，可考虑扩展清洗范围
3. **尝试更大基座**：RoBERTa-large (355M) 冲击 0.65+ Macro F1
4. **不对称 dropout**：对 RoBERTa 层保持 0.1，对 BiLSTM/Classifier 提升至 0.2~0.3
5. **Label Smoothing**：添加 eps=0.1，进一步缓解"悲伤吸星"效应
