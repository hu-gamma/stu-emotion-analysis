# 今日工作进度报告（2026-04-23）

## 三、消融实验：去掉 Dreaddit 训练数据

### 3.1 消融模型训练
- 训练数据：OCEMOTION + ECSA（去掉 dreaddit_train，共 35967条）
- 模型架构：与完整模型完全一致（RoBERTa-BiLSTM，BiLSTM 保留）
- 训练配置：5 epochs, lr=1e-5, CrossEntropyLoss + 类别权重 + WeightedRandomSampler
- 训练历史：
  - Epoch 1: Train Acc=42.30%, Val Acc=49.33%, Val Macro F1=0.2866
  - Epoch 2: Train Acc=62.62%, Val Acc=51.36%, Val Macro F1=0.3103
  - Epoch 3: Train Acc=69.86%, Val Acc=52.81%, Val Macro F1=0.3255
  - Epoch 4: Train Acc=73.97%, Val Acc=54.76%, Val Macro F1=0.3259
  - Epoch 5: Train Acc=78.91%, Val Acc=57.12%, Val Macro F1=0.3313
- 模型保存：/mnt/checkpoints/roberta_bilstm_no_dreaddit/best_model.pt

### 3.2 消融模型评估
- 测试集：dreaddit_test_processed.csv（715条，与完整模型完全一致）
- Accuracy：44.48%
- Macro F1：0.2480
- Weighted F1：0.4443
- 平均推理延迟：9.36 ms/条
- 预测文件：/mnt/results/compare/roberta_no_dreaddit_predictions.jsonl

### 3.3 消融实验结论
- 去掉 Dreaddit 后，测试准确率反而提升 +4.62%（39.86% -> 44.48%）
- Weighted F1 提升 +10.49%（0.3394 -> 0.4443）
- Dreaddit 训练数据对当前测试任务存在负迁移，翻译质量可能是主因
- 核心大类（悲伤、愤怒、快乐）F1 显著提升，但部分小类（好奇、感激、期待）下降
- 两模型预测不一致率高达 78.2%，分歧场景中无 Dreaddit 模型更优（194 vs 161）
